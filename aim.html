Aim: To create a tavel guide app

We aim to create an app which allows the user to search, browse and plan trips. This application should put the user’s job of researching airline tickets, accommodation, location and activities all into one instead of having to creating a custom itinerary which can be a complex and time-consuming task. We hope that this application will shorten the time is takes for users to plan and book a holiday that fulfils their requirements and keeps track of all expenses. 

Goal 1: Create plans – key path scenarios, wireframes, prototype 

By creating Key paths scenarios and wireframes, we can determine the layout and how the user will progress through application as they browse/book a trip through the ‘Smile Travel’ application. A protype is also planned to be created so a user can interact with a functioning version of the application to determine if the usability of the program is up to par or if improvements are required. 

Goal 2: Create a system to suggest locations and activities to the user based on their previous interactions on the app. 

By creating a system that suggest locations and activities for users, this will incentivise these users to further navigate the application and potentially provide an opportunity for them to find a destination that they wish to travel to. 

Goal 3: Create system to book

If users plan to go on multiple trips, the application should have the ability to book the necessary requirements of a holiday, that being airline tickets accommodation, and any other activities that are offered on the platform. By being able to keep track of the booking information, the users can identify which destinations/airline tickets and activities are related to which holiday booked. 

Goal 4: upload to app store / play store 

By uploading this application to the app store and google play store, this will give the application the greatest amount of exposure and ease in terms on downloading and running the application within current day IT technologies.
